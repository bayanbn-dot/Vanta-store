# Vanta | Modern Interactive E-commerce Landing Page 

Vanta is a minimalist and highly interactive fashion landing page built with a focus on modern UI/UX principles. This project showcases smooth entrance animations, a custom earth-tone color palette, and a fully responsive layout.

## ✨ Key Features
* **Dynamic UI Interactions:** Custom JavaScript logic for toggleable search, cart, and user menus.
* **Scroll-Responsive Header:** Features a dynamic shadow effect that activates on scroll for a premium feel.
* **Entrance Animations:** Integrated JavaScript animations that bring elements to life as the user interacts with the page.
* **Interactive Product Slider:** Powered by **Swiper.js** for a seamless "New Arrivals" browsing experience on touch and desktop.
* **Fully Responsive:** Optimized with CSS Grid and Flexbox for all screen sizes (from 360px mobile to ultra-wide desktops).
* **Custom Design System:** Utilizes CSS variables for a consistent and easily maintainable color identity.

## 🛠️ Built With
* **HTML5:** Semantic structure for better SEO and accessibility.
* **CSS3:** Advanced Grid layouts, Flexbox, and Custom Properties.
* **Vanilla JavaScript:** Clean, dependency-free code for DOM manipulation and event handling.
* **Swiper JS:** For high-performance touch sliders.

## 🚀 How to Launch
1. Clone the repository: `git clone https://github.com/your-username/vanta.git`
2. Open `index.html` in your favorite browser.
3. Scroll and interact with the icons to experience the dynamic animations.
